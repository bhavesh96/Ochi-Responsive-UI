# Ochi Style Interactive Website – Frontend Implementation

This project is a modern, visually engaging frontend experience built with React and Tailwind CSS.  
It focuses on smooth animations, clean typography, interactive elements, and a polished UI inspired by contemporary creative-agency websites.

---

## Features

### Fully Responsive  
The layout is optimized for all screen sizes — mobile, tablet, and desktop.

### Responsive Navigation  
A clean, minimal, and responsive navbar with hover animations and fluid transitions.

### Modern Hero Section  
Large typography with smooth transitions and a strong visual hierarchy.

### Animated Marquee Section  
A bold scrolling text section using custom styling to create dynamic movement on the page.

### Eye-Tracking Interaction  
A custom interactive “eye-tracking” component where both eyes follow the user’s cursor using trigonometry (`atan2`) for precise rotation.


### Fast & Lightweight  
- Built with **React**
- Styled using **Tailwind CSS**
- Indexed asset loading & optimized performance
